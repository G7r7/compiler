# C-- Compiler

This project is a simple c compiler limited to the following tokens :
- Constant (integer)
- Identifier (variable name)
- \+
- \-
- \*
- /
- %
- !
- & (address)
- = (initialize)
- ==
- !=
- <
- <=
- \>
- \>=
- &&
- ||
- ,
- ;
- int
- if
- else
- for
- while
- do
- break
- continue
- return
- (
- )
- [
- ]
- {
- }
- EOF (End Of File)
